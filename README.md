# VueEndTest 后台管理系统

Spring Boot构建的后端服务，提供RESTful API接口支持

## 技术栈
- **核心框架**: Spring Boot 3.4.3
- **持久层**: MyBatis + Spring Data JPA
- **数据库**: MySQL 8.0
- **工具链**:
    - Lombok 1.18.28（实体类简化）
    - Maven 3.8.1+
    - JDK 17
